🧮 Алгебра

Формулы сокращённого умножения

(a+b)² = a² + 2ab + b²
(a−b)² = a² − 2ab + b²
a² − b² = (a−b)(a+b)
(a+b)³ = a³ + 3a²b + 3ab² + b³
(a−b)³ = a³ − 3a²b + 3ab² − b³
a³ + b³ = (a+b)(a² − ab + b²)
a³ − b³ = (a−b)(a² + ab + b²)

Арифметическая прогрессия

aₙ = a₁ + (n−1)d
Sₙ = (a₁ + aₙ)·n / 2 = (2a₁ + (n−1)d)·n / 2
Характеристическое свойство: aₙ = (aₙ₋₁ + aₙ₊₁)/2

Геометрическая прогрессия

bₙ = b₁·qⁿ⁻¹
Sₙ = b₁·(qⁿ − 1)/(q − 1), q ≠ 1
Сумма бесконечной убывающей: S = b₁/(1−q), |q|<1
Характеристическое свойство: bₙ² = bₙ₋₁·bₙ₊₁

Свойства степеней

aᵐ·aⁿ = aᵐ⁺ⁿ
aᵐ / aⁿ = aᵐ⁻ⁿ, a ≠ 0
(aᵐ)ⁿ = aᵐⁿ
(ab)ⁿ = aⁿ·bⁿ
(a/b)ⁿ = aⁿ / bⁿ, b ≠ 0
a⁰ = 1, a ≠ 0
a⁻ⁿ = 1/aⁿ, a ≠ 0

Свойства корней

√(ab) = √a·√b, a,b ≥ 0
√(a/b) = √a / √b, a ≥ 0, b > 0
(√a)² = a, a ≥ 0
√(a²) = |a|
ⁿ√(aᵐ) = aᵐ⁄ⁿ, a ≥ 0

Свойства логарифмов

logₐ(xy) = logₐx + logₐy
logₐ(x/y) = logₐx − logₐy
logₐxᵏ = k·logₐx
logₐb = log_c b / log_c a (формула перехода)
a^(logₐb) = b (основное тождество)
logₐa = 1, logₐ1 = 0

Квадратные уравнения

ax² + bx + c = 0
D = b² − 4ac
x₁,₂ = (−b ± √D) / (2a)
Если D > 0: два корня
Если D = 0: один корень
Если D < 0: нет корней
Теорема Виета: x₁ + x₂ = −b/a, x₁·x₂ = c/a

Разложение на множители

ax² + bx + c = a(x − x₁)(x − x₂)
Вынесение общего множителя
Метод группировки
Использование ФСУ

📐 Планиметрия

Площадь треугольника

S = ½·a·hₐ (через основание и высоту)
S = ½·ab·sin γ (через две стороны и угол)
S = √(p(p−a)(p−b)(p−c)) (формула Герона)
S = abc/(4R) (через радиус описанной окружности)
S = pr (через радиус вписанной окружности)

Теорема синусов

a/sin A = b/sin B = c/sin C = 2R
где R — радиус описанной окружности

Теорема косинусов

a² = b² + c² − 2bc·cos A
Используется для нахождения стороны или угла

Прямоугольный треугольник

Теорема Пифагора: a² + b² = c²
Высота из прямого угла: h² = a₁·b₁
Радиус вписанной окружности: r = (a+b−c)/2
Площадь: S = ab/2
Медиана к гипотенузе: m = c/2

Равносторонний треугольник

Высота: h = a√3/2
Площадь: S = a²√3/4
R = a√3/3 (описанная окружность)
r = a√3/6 (вписанная окружность)

Четырёхугольники

Прямоугольник: S = ab, P = 2(a+b)
Квадрат: S = a², d = a√2
Параллелограмм: S = a·hₐ = ab·sin α
Ромб: S = a·h = ½·d₁·d₂
Трапеция: S = (a+b)/2 · h

Окружность и круг

Длина окружности: C = 2πR
Площадь круга: S = πR²
Длина дуги: L = πR·α/180°
Площадь сектора: S = πR²·α/360°

Длина биссектрисы

l = (2ab·cos(γ/2))/(a+b)
или l = √(ab − a₁b₁),
где a₁, b₁ — отрезки стороны

Длина медианы

mₐ = ½√(2b² + 2c² − a²)
Медиана делит сторону пополам

🧊 Стереометрия

Куб, призма, параллелепипед

Объём призмы: V = Sосн·h
Прямоугольный параллелепипед: V = abc
Куб: V = a³, Sпов = 6a²
Призма: Sполн = Sбок + 2Sосн

Цилиндр и конус

Цилиндр: V = πR²h
Sбок = 2πRh, Sполн = 2πR(R+h)
Конус: V = ⅓πR²h
Sбок = πRl, где l = √(R²+h²)

Пирамида и шар

Пирамида: V = ⅓Sосн·h
Шар: V = ⁴⁄₃πR³
Площадь поверхности шара: S = 4πR²

📐 Тригонометрия

Основные тождества

sin²α + cos²α = 1
tgα = sinα/cosα
ctgα = cosα/sinα
tgα·ctgα = 1
1 + tg²α = 1/cos²α
1 + ctg²α = 1/sin²α

Формулы сложения

sin(α±β) = sinα·cosβ ± cosα·sinβ
cos(α±β) = cosα·cosβ ∓ sinα·sinβ
tg(α±β) = (tgα ± tgβ)/(1 ∓ tgα·tgβ)

Формулы двойного угла

sin2α = 2sinα·cosα
cos2α = cos²α − sin²α
cos2α = 2cos²α − 1
cos2α = 1 − 2sin²α
tg2α = 2tgα/(1−tg²α)

Сумма в произведение

sinα + sinβ = 2sin((α+β)/2)·cos((α−β)/2)
sinα − sinβ = 2cos((α+β)/2)·sin((α−β)/2)
cosα + cosβ = 2cos((α+β)/2)·cos((α−β)/2)
cosα − cosβ = −2sin((α+β)/2)·sin((α−β)/2)

Таблица значений

0°: sin=0, cos=1, tg=0
30°: sin=½, cos=√3/2, tg=√3/3
45°: sin=√2/2, cos=√2/2, tg=1
60°: sin=√3/2, cos=½, tg=√3
90°: sin=1, cos=0, tg не опр.

🎲 Теория вероятностей

Основные формулы

P(A) = m/n (классическая вероятность)
P(A) + P(¬A) = 1
P(A∪B) = P(A) + P(B) − P(A∩B)
P(A∩B) = P(A)·P(B) (для независимых)
Формула Бернулли: Pₙ(k) = Cₙᵏ·pᵏ·qⁿ⁻ᵏ

📈 Производные и интегралы

Правила дифференцирования

(c)' = 0
(xⁿ)' = n·xⁿ⁻¹
(u ± v)' = u' ± v'
(uv)' = u'v + uv'
(u/v)' = (u'v − uv')/v²
(sin x)' = cos x
(cos x)' = −sin x
(tg x)' = 1/cos²x
(eˣ)' = eˣ
(aˣ)' = aˣ·ln a
(ln x)' = 1/x

Первообразные

∫xⁿ dx = xⁿ⁺¹/(n+1) + C, n ≠ −1
∫1/x dx = ln|x| + C
∫sin x dx = −cos x + C
∫cos x dx = sin x + C
∫eˣ dx = eˣ + C
∫aˣ dx = aˣ/ln a + C
